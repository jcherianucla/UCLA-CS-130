    func Authenticate ¶ 

    func Authenticate(next http.HandlerFunc) http.Handler

    Authenticate wraps any route handler with authentication capability to ensure that to enter those routes, the caller has to be authenticated via JWT. It retuns a new handler with authentication handled.

    func Logging ¶

    w http.ResponseWriter,
    r *http.Request,
    next http.HandlerFunc)

    Logging adds some local logging between each request. It takes the response writer and request to pass through to the next handler.
